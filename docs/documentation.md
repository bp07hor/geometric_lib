# GEOMETRIC_LIB
В данной библиотеке **geometric_lib** собраны функции реализующие подсчёт *площади* а также *периметра* при помощи языка программирования *python*.
Данные функции выводят необходимый пользователю результат, принимая необходимые для подсчета показатели, такие как:
- Радиус окружности
- Стороны фигур
- Высоты фигур

## Circle.py
Данная функция реализует подсчёт *площади* и *периметра* (длинны) окружности, принимая показатель r, который является *радиусом окружности*.

Подсчёт *площади* происходит при помощи встроенной функции **area**:
> def area(r):
>    return math.pi * r * r
Пример вызова:
> area(5)
Вывод:
> 78.53981634

Подсчёт *периметра* (длинны окружности) происходит при помощи встроенной функции **perimeter**:
> def perimeter(r):
>    return 2 * math.pi * r
Пример вызова:
> perimeter(5)
Вывод:
> 31.41592654

## Rectangle.py
Данная функция реализует подсчёт *площади* и *периметра* прямоугольника, принимая показатели a и b, которые являются *сторонами прямоугольника*.

Подсчёт *площади* происходит при помощи встроенной функции **area**:
> def area(a, b):
>   return a * b
Пример вызова:
> area(2, 3)
Вывод:
> 6

Подсчёт *периметра* происходит при помощи встроенной функции **perimeter**:
> def perimeter(a, b):
>   return 2 * (a + b)
Пример вызова:
> perimeter(2, 3)
Вывод:
> 10

## Square.py
Данная функция реализует подсчёт *площади* и *периметра* квадрата, принимая показатель a, который является *стороной квадрата*.

Подсчёт *площади* происходит при помощи встроенной функции **area**:
> def area(a):
>   return a * a
Пример вызова:
> area(1)
Вывод:
> 1

Подсчёт *периметра* происходит при помощи встроенной функции **perimeter**:
> def perimeter(a):
>   return 4 * a
Пример вызова:
> perimeter(1)
Вывод:
> 4

## Triangle.py
Данная функция реализует подсчёт *площади* и *периметра* треугольника, принимая показатели a, b, c и h, где a, b и c являются *сторонами треугольника*, а h является *высотой треугольника падающей на сторону a*.

Подсчёт *площади* происходит при помощи встроенной функции **area**:
> def area(a, h):
>   return a * h / 2
Пример вызова:
> area(15, 20)
Вывод:
> 150

Подсчёт *периметра* происходит при помощи встроенной функции **perimeter**:
> def perimeter(a, b, c):
>   return a + b + c
Пример вызова:
> perimeter(15, 20, 25)
Вывод:
> 60
